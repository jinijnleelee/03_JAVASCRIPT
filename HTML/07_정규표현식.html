정규표현식(Regular Expression)
특정한 규칙을 가진 무자열 집합을 표현하는데 사용하는 형식 언어
정규표현식을 이용하면
입력문 문자열 에 대한 특정 조건 검색,일치포함여부,
자판에 대한 조건문을 간단히 처리할수있음

정규표현 참고사이트 

정규표현식이 참고 사이트 

* 수량관련 메타 문자 
a{s} : a문자가 5게 존재 === aaaaa
a{2,5} : a가 2개 이상 5개 이하 == aa,aaa,aaaa,aaaaa
a{2,} : a가 2개 이상 
a{,5} : a가 5개 이하

* : 0개 이상 == 0번 이상 반복 == 있어도 되고 없어도 되고
+ : 1개 이상 == 1번 이상 반복
? : 0개 또는 1개 
, : 1칸 (개행문자 제외한 문자 하나)

이름 유효성 검사

    - 한글 2글자 이상 6글자 이하의 문자열
    - 단자음 / 모음 경우는 제외
    - 우요한 경우 초록섹 글자로 "유효한 이름 형식입니다 " 출력
    - 유효하지 않은 경우 빨간색 글씨로 "잘못된 형식입니다" 출력
